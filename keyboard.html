Rekomendasi Keyboard

Keyboard adalah salah satu perangkat keras komputer yang sangat penting bagi sebagian besar pengguna. Bagi sebagian orang, keyboard adalah alat utama untuk mengetik dokumen, bermain game, atau bahkan bekerja dalam bidang kreatif. Namun, memilih keyboard yang sesuai dengan anggaran bisa menjadi tugas yang cukup menantang. Untungnya, ada banyak pilihan keyboard berkualitas dengan harga terjangkau yang dapat memenuhi kebutuhan Anda. Keyboard mechanical telah menjadi pilihan yang populer di kalangan pecinta perangkat keras komputer. Mereka dikenal karena ketahanan, daya tahan, dan sensasi mengetik yang unik. Maka dari itu disini saya akan berikan rekomendasi keyboard mechanical dengan harga terjangkau.

1. Fantech ATOM96 MK890

Fantech ATOM96 MK890 memiliki desain yang cukup menarik. Keyboard ini memiliki layout tenkeyless, yang berarti tidak ada numpad di sebelah kanan. Ini membuatnya lebih kompak dan cocok untuk para pengguna yang ingin lebih banyak ruang untuk pergerakan mouse atau para pengguna dengan meja sempit. Keyboard ini memiliki bezel yang cukup ramping dan pencahayaan RGB yang menarik. Keycaps pada keyboard ini terbuat dari bahan yang terasa kuat dan tahan lama. Keyboard ini juga memiliki font yang jelas dan mudah dibaca, meskipun beberapa orang mungkin lebih suka opsi keycap dengan font yang lebih sederhana. Anda juga mendapatkan tombol knob untuk media dan kontrol pencahayaan langsung di atas tombol fungsi, yang memudahkan penggunaan.

  • Spesifikasi & Harga
  • • Type of Switch : Mechanical
  • • Switch Color : Blue / Red
  • • Total Keys 96
  • • Anti-ghosting Full-keys
  • • Keycaps Double-injection
  • • Core Construction ABS Plastic
  • • Berat : 648g
  • • Ukuran : 303.2 x 104.3 x 45mm
  • • Software Driver : No
  • • Harga: Rp.250.000-Rp.325.000
2. Fantech ATOM MK886

Fantech ATOM MK886 ini adalah keyboard ukuran penuh (full size form factor keyboard) yang menyediakan semua tombol yang diperlukan pengguna termasuk tombol numerik dan simbol. Susunan tombol yang lengkap akan memudahkan pengetikan dalam situasi apa pun. Karena pengguna tidak perlu mengingat penekanan tombol untuk mengaktifkan fungsi yang diinginkan.

  • Spesifikasi & Harga
  • • Type of Switch : Mechanical
  • • Switch Color : Blue / Red
  • • Total Keys 104
  • • Anti-ghosting 25 keys
  • • Keycaps Double-injection
  • • Core Construction ABS Plastic
  • • Berat : 700g
  • • Ukuran : 435 x 127 x 32mm
  • • Software Driver : No
  • • Harga: Rp.225.000-Rp.260.000
3. Fantech ATOM MK876 TKL

Fantech ATOM MK876 merupakan seri TKL (Ten-Keyless) dari Fantech ATOM MK886. Perbedaannya terletak pada ukuran Keyboard yang lebih kecil dengan menghilangkan numpad sehingga cocok untuk kalian yang membutuhkan ruang yang lebih luas untuk pergerakan mouse atau memiliki meja yang sempit.

  • • Core Construction ABS Plastic
  • • Berat : 565g
  • • Ukuran : 355 x 127 x 32mm
  • • Software Driver : No
  • • Harga: Rp.180.000-Rp.250.000
